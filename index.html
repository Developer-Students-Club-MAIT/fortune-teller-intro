Project Description

Zoltar Is an artificial conversation entity or an AI chatbot for short. The project was created using NLTK framework of the python programming language. The main driving force behind this project is the custom made dataset. The dataset was created using data from the New York Times horoscope section and by scraping comments from relevant sub Reddits.
